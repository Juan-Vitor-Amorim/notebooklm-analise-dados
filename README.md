## 📌 Sobre o projeto

Este projeto foi desenvolvido utilizando o NotebookLM como ferramenta de aprendizagem ativa para estudar análise de dados.

O objetivo foi reunir materiais sobre a área, testar prompts, consolidar conceitos importantes e criar um guia de estudos para quem deseja ingressar no mercado em até seis meses.

---

## 🎯 Objetivos

- Aprender o processo completo de análise de dados;
- Entender o fluxo ETL;
- Dominar Excel, SQL, Power BI e Python;
- Criar dashboards e relatórios;
- Desenvolver projetos para portfólio;
- Conhecer as diferenças entre Data Analytics, BI e Data Science.

## 🤖 Prompts utilizados

### Prompt 1

> Explique o processo completo de análise de dados utilizando apenas as fontes fornecidas.

### Principais etapas encontradas

- Definição do problema;
- Extração dos dados;
- Limpeza e transformação;
- Análise;
- Visualização;
- Tomada de decisão.

---

### Prompt 2

> Qual a diferença entre Data Analytics, Data Science e Business Intelligence?

#### Data Analytics

Transforma dados em informações úteis.

#### Business Intelligence

Cria dashboards e relatórios estratégicos.

#### Data Science

Utiliza Machine Learning e modelos avançados.

---

### Prompt 3

> Explique ETL de forma simples.

ETL significa:

- Extract (Extrair);
- Transform (Transformar);
- Load (Carregar).
